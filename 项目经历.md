# 苍穹外卖（CangQiong Takeout）

## 项目介绍

基于 Spring Boot 3 + MyBatis 的前后端分离外卖点餐系统，采用 Maven 多模块架构（parent/common/pojo/Server），分为管理端和用户端两套接口。管理端提供员工管理、菜品管理、套餐管理、分类管理、订单管理、数据统计与报表导出等功能；用户端基于微信小程序，支持微信登录、浏览菜品、购物车、下单支付、催单、历史订单等功能。项目集成 Redis 缓存、WebSocket 实时通信、Spring Task 定时任务、阿里云 OSS 文件存储等中间件，贴近企业级外卖业务场景。

## 技术栈

- Java 21、Spring Boot 3.5
- MyBatis + PageHelper 分页插件
- MySQL 数据库
- Redis（Spring Data Redis）
- WebSocket（Spring Boot Starter WebSocket）
- Spring Task 定时任务
- JWT（jjwt）鉴权认证
- 阿里云 OSS 对象存储
- Apache POI（Excel 报表导出）
- Apache HttpClient（微信接口调用）
- Swagger / Knife4j 接口文档
- Lombok
- FastJSON
- Spring AOP（自定义注解 + 切面）
- Maven 多模块管理

## 项目亮点

1. **自定义注解 + AOP 自动填充公共字段**：通过 `@AutoFill` 注解和切面拦截 Mapper 层方法，利用反射自动填充 createTime/updateTime/createUser/updateUser，消除重复代码。

2. **JWT 双端鉴权体系**：管理端和用户端分别配置独立的 JWT 拦截器，结合 ThreadLocal 存储当前用户 ID，实现接口级权限隔离。

3. **Redis 缓存店铺营业状态**：店铺状态存入 Redis，管理端和用户端共享缓存，避免频繁查库，提升高并发读取性能。

4. **WebSocket 来单提醒与客户催单**：通过 WebSocket 长连接实现支付成功后服务端主动推送来单提醒，以及用户催单实时通知商家。

5. **Spring Task 定时处理异常订单**：定时扫描超时未支付订单自动取消，长时间派送中订单自动完成，保障订单生命周期状态准确。

6. **微信小程序登录**：调用微信 jscode2session 接口获取 openid 实现授权登录，新用户自动注册并返回 JWT Token。

7. **阿里云 OSS 文件上传**：集成阿里云 OSS Starter 实现菜品图片云端存储，文件服务与应用解耦。

8. **Apache POI 导出运营报表**：基于 Excel 模板填充近 30 天营业额、订单、用户等运营数据并提供下载。

9. **全局异常处理与统一响应封装**：通过 `@RestControllerAdvice` 捕获业务异常和 SQL 约束冲突，配合统一 Result 响应体规范前后端交互。

10. **Maven 多模块分层架构**：拆分为 parent/common/pojo/Server 四模块，职责清晰、依赖明确，符合企业级工程化规范。
